How to use

1. Setup your HTML

jQuery Sliding Deck can be applied on any element that has at least 2 children. You can use it to display a small page element or all your page content.

Here is an example of what your page content might look like:


<main>
  <section>
    <h2>Section 1</h2>
    <p>Section content</p>
  </section>
  <section>
    <h2>Section 2</h2>
    <p>Section content</p>
  </section>
</main>

2. Include jQuery with the plugin files

Next, you need to include jQuery (version 3.1.1 or greater) and the JavaScript and CSS file of jQuery Sliding Deck as follows...

In the <head> tag:


<!-- jQuery Sliding Deck -->
<link rel="stylesheet" href="jquery.slidingDeck.min.css" />

Before the </body> tag:


<!-- jQuery 3.1.1+ -->
<script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/3.1.1/jquery.min.js"></script>

<!-- jQuery Sliding Deck -->
<script src="jquery.slidingDeck.min.js"></script>

3. Call the plugin

Lastly, activate jQuery Sliding Deck by calling the slidingDeck function on your parent element (e.g. <main>) ‚Äî you can change the default options at the same time:


$(document).ready(function() {
  $('main').slidingDeck([options]);
});

Customising

1. Options

You can use any or all of these options when initialising jQuery Sliding Deck:

Option Default Type Description Options 
cards
 "section" jQuery path selector The path to the child elements 
height
 "full" string/int The height that jQuery Sliding Deck should fill full: full height of the parent element
auto: the cards will take their natural (default) height
[integer]: fix the height in pixels		 
zIndexBase
 10 int The z-index for the 1st card ‚Äî used as a multiplier for all subsequent cards 
classes: {
  cardBase,
  cardActive,
  cardFixed,
  cardPrevious,
  cardNext,
  cardGhost,
}
 cardBase: "jsd-card"
cardActive: "jsd-card--active"
cardFixed: "jsd-card--fixed"
cardPrevious: "jsd-card--previous"
cardNext: "jsd-card--next"
cardGhost: "jsd-card--ghost" 		associative array The CSS class names used by jQuery Sliding Deck

2. Defaults

All default options:


$(document).ready(function() {
  $("section").slidingDeck({
    cards: 'section',
    height: 'full',
    zIndexBase: 10,
    classes: {
      cardBase: 'jsd-card',
      cardActive: 'jsd-card--active',
      cardFixed: 'jsd-card--fixed',
      cardPrevious: 'jsd-card--previous',
      cardNext: 'jsd-card--next',
      cardGhost: 'jsd-card--ghost',
    }
  });
});
